Aplikasi ini merupakan aplikasi untuk enkripsi pesan. untuk menggunakannya, ikuti Langkah berikut:

1. Siapkan file yang berisi pesan yang akan dienkripsi. untuk saat ini, hanya file txt yang dapat digunakan. jadi, buat file txt baru, kemudian tuliskan pesan anda ke dalam file tersebut. misalkan nama file nya adalah contoh-teks.txt
2. jalankan program window.py. program ini akan membuka window apikasi.
3. Pada ruas kiri aplikasi, terdapat symbol upload. klik simbol tersebut, kemudian pilih file txt tadi yang ingin anda upload, misalnya file contoh-teks.txt
4. tuliskan password pada ruas kanan aplikasi. pastikan password hanya terdiri dari huruf, tanpa spasi, angka ataupun karakter lain.
5. klik tombol encrypt
6. jika proses enkripsi berhasil, akan muncul status `OK - file anda dienkripsi dengan aman`. untuk melihat hasilnya, buka lagi folder dimana file window.py berada. file hasil enkripsi anda akan memiliki nama file dengan akhiran `- encrypted`. jika file teks asli anda Bernama contoh-teks.txt, maka file hasil enkripsi anda akan Bernama `contoh-teks - encrypted.txt`.
